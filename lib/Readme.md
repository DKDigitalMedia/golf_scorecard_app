# Golf Scorecard App

A Flutter app to track golf rounds, holes, and courses with tee boxes, approach/putt tracking, and round summaries. Built using **Flutter**, **Drift** for local database, and **Riverpod** for state management.

---

## **Features**

- **Main Menu**
  - Start a new round
  - View saved rounds
  - Create/edit courses and tees

- **Course Editor**
  - Create or edit courses
  - Add multiple tee boxes with yardage
  - Set course rating and slope

- **Hole Entry**
  - Enter score, putts, penalties, FIR/GIR/Up-and-down
  - Adjustable approach and first putt distances
  - Select approach location with colored chips:
    - Center → green
    - Left/Right → yellow
    - Long/Short → cyan
  - Automatic save when navigating with Back/Next
  - Next button shows "Round Summary" on 18th hole
  - Maximum approach distance adjusts based on selected tee yardage

- **Round Summary**
  - Displays all 18 holes
  - Shows score, putts, FIR, GIR, and Up-and-Down
  - Highlights completed holes

- **Saved Rounds & Dashboard**
  - Lists all saved rounds
  - Displays course and completion status
  - Tap to resume or view holes

## **Setup**

1. **Install dependencies**

flutter pub get

2. ** Generate Drift database code

flutter pub run build_runner build --delete-conflicting-outputs


3. ** Run the app

flutter run
